Om kursen webtec

Teknisk webbdesign och användbarhet

Kursen webtec är en grundkurs i teknisk webbdesign som fokuserar på att lära studenterna att bygga professionella och användarvänliga webbsidor. Som förstaårsstudent på BTH:s webbutvecklarprogram är detta min första riktiga djupdykning i världen av webbutveckling, och det känns otroligt spännande att äntligen få lära mig grunderna på riktigt!

Kursen täcker allt från grundläggande HTML och CSS till mer avancerade koncept som responsiv design, tillgänglighet och användarupplevelse. Vi lär oss att tänka på webben inte bara som en teknisk plattform, utan som ett medium där design, funktion och användarupplevelse måste samarbeta för att skapa något verkligt värdefullt. Det är här som min tidigare erfarenhet av programmering i C++ och Java börjar kombineras med kreativitet och designtänk på ett helt nytt sätt.

Genom olika kursmoment bygger vi upp kompetensen steg för steg - från att skapa enkla statiska sidor till att förstå komplexa layoutsystem och interaktiva element. Kursen lägger stor vikt vid att lära oss webbutveckling "på rätt sätt" - med korrekt semantisk markup, tillgänglighet i fokus och med förståelse för webbläsares funktionalitet och begränsningar.

Det som gör kursen särskilt intressant är balansen mellan teknisk kunskap och kreativ problemlösning. Vi lär oss inte bara att skriva kod som fungerar, utan att skapa digitala upplevelser som är både vackra och funktionella. Som någon som kommer från mer traditionell programmering uppskattar jag verkligen hur kursen kombinerar logiskt tänkande med visuell design och användarcentrerat fokus.

Webbutveckling - där teknik möter kreativitet för att skapa digitala upplevelser.
